# PostgreSQL Tips

## Quickstart Guide
Basic use tips for using PostgreSQL to do simple database management.  

This is intended to be a "quick-start" guide to navigating Postgres in the console with the goal of being able to bring old data up into new table and databases.  In our environment we have a lot of old data that needs to be consolidated onto new databases so the old servers can be dispositioned.  The other goal was to use the most basic and simple commands trying to avoid extra software and training time.  There may be much more effective or efficient ways to do this, but this is also intended to maximize simplicity and compatibility as well.

Data set used for examples is the EV population of the State of Washington through data.gov
